## 👨‍💻 About me

I lead the **Playvox Cloud Team @ NICE**, driving **CI/CD, Cloud (AWS), SRE and FinOps** across teams. My job isn't shipping infrastructure for the sake of it. It's **partnering with product** so the platforms we own actually move the needle: more **revenue**, better **user experience**, lower **cost per outcome**.

But the job title is just one slice. I'm also an **educator**, a **community organizer**, a **football fan**, a **traveler**, a **bookworm** (rarely tech), and a **lifelong Linux + penguins enthusiast** 🐧.

> *"Infrastructure is invisible when it works. The win is what product can ship on top of it."*

---

## 🧭 What I do today

| Area | What that looks like |
|------|----------------------|
| 🤝 **Product × Engineering** | Sitting next to product to turn features into reliable, revenue-driving experiences. Infra is a means, not the goal. |
| ☁️ **Cloud (AWS)** | Multi-account architectures, IaC with Terraform, secure-by-default networking, identity with OIDC. |
| 🔁 **CI/CD** | Pipelines across **CircleCI, GitLab CI, Buildkite, Drone & GitHub Actions**. Golden paths and OIDC-based deploys. |
| 🛡️ **SRE** | SLOs tied to user experience, error budgets, observability, incident response, postmortems that actually change behavior. |
| 💸 **FinOps** | Cost visibility, rightsizing, savings plans. Cost as a first-class engineering metric next to latency and uptime. |
| 🔐 **Security** | Backed by an **Information Security degree (UOC, Spain)**. Security woven into pipelines, identity and infra, not bolted on. |
| 🧑‍🏫 **Leadership** | Leading engineers, growing capability, balancing roadmap vs. reliability vs. spend. |
| 🤖 **AI for Ops** | Exploring how **Claude Code** and AI agents can level up everyday **SRE / DevOps** work. |
